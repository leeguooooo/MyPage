关于我们

去哪儿网移动架构组（YMFE）是由 FE，iOS 和 Android 工程师共同组成的，去哪儿最具想象力、创造力和影响力的大前端团队

我们致力于为业务开发提供一体化的移动开发解决方案，努力提升各个产品线在移动端的开发效率及使用体验

我们在做什么

目前，我们产出的框架和基础服务已运行在去哪儿网无数项目之中。

不仅仅如此，我们还希望持续改进和完善这些解决方案，以期帮助各个业务线打造出最好的产品：

  • 公司移动解决方案架构设计，核心框架代码开发；
  • 业务辅助平台和效率工具开发；
  • 基础服务及其他公司通用的工具服务的开发和维护；
  • 前沿技术调研、开发方式创新；
  • 还有很多。。。

技术框架

  • React 技术栈：QRN、QReact、React Web 等
  • Hybrid 技术：Hy2、Yo等
  • Avalon、小程序等

工具与平台

  • 效率工具：YApi、YShow、Ydoc等
  • 构建部署：YKit、QP热更新等
  • YStats、开放平台等

其他项目

  • 客户端调试、静态资源管理等

我们的风格:

  • 不仅仅是工程师，更是团队的建设者。
  • 你可以参与每一个项目，只要你能贡献你的力量。
  • 我们不会很明确地要求你做什么、怎么做，你需要自己思考去做什么、怎么做，给你更大的创新空间。
  • 我们有自己的技术产品和用户，人人都是 PM，需要主动发现并解决遇到的问题，或者提出更好的方案并实现。
  • 我们不在意你的学历和过去，只在意你现在的理想。
