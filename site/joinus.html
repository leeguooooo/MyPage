去哪儿网移动架构组（ YMFE ）2017 首次社招

此次招聘的职级为：q14-q18（ 对应之前的级别为：12-14 ）

高级/资深前端工程师（Open）

我们可能会需要你做以下这些事情：

  • 移动端解决方案架构设计，核心框架代码开发。例如 Hybrid、React Native 相关工作等；
  • 改进和优化现有的前端工具链和前端开发流程；
  • 效率工具及通用服务开发；
  • 前沿技术调研、开发方式创新；

职位要求

  • 敏锐的技术敏感度，对技术有饱满的热情；
  • 有 Hybrid, React, React Native, Vue 等时髦技术的开发经验；
  • 有丰富的 Web 标准、易用性、浏览器端原理以及网站性能优化等方面的知识；
  • 有独立的产品意识，能负责一个完整的项目技术选型及实施；
  • 有团队协作精神，善于学习，乐于探索，不墨守陈规；
  • 4年以上的前端开发经验；

高级/资深 iOS 工程师（Open）

你可能要面临这些挑战：

  • 负责 React Native，Hybrid 等框架 iOS 端的研发和探索；
  • 协同前端以及 Android 团队，独立完成框架的设计与开发；
  • 不断在可用性，可维护性和性能等方面做到极致；

职位要求

  • 精通 Objective-C/C/C++，具有良好的编码规范，较强 Debug 能力；
  • 熟练掌握 iOS 各个 UI 组件的使用，并能够高度自定义 UI 组件；
  • 熟练掌握 block、runtime、GCD 等相关知识；
  • 熟悉 App 内存占用、耗电、网络、UI 渲染等性能方面分析和优化的方法；
  • 扎实的数据结构算法基础，熟悉面向对象方法和常用设计模式；
  • 熟悉 JavaScript 等其他语言；
  • 有独立解决问题的能力和很强的学习能力；
  • 具备优秀的沟通和协调能力，良好的服务精神及团队合作能力，责任心强；
  • 具有大型 APP 开发经验，APP 上架作品，熟悉 Python，熟悉 Android 开发，服务端开发经验，做过框架 SDK 等相关工作者优先；
  • 具有较强的文档编写能力（有原创 blog 者优先）；
  • 有 React Native，Hybrid，Weex 等开发经验优先；
  • 4年以上 iOS 平台软件开发经验；

高级/资深 Android 工程师（Open）

这些点可能是你需要关注的：

  • 负责 Hybrid、React Native 等框架 Android 端的研发和探索；
  • 协同前端以及 iOS 团队，独立完成框架的设计与开发；
  • 及时掌握新的 Android 相关的技术点。

职位要求

  • 了解操作系统原理，掌握常见的数据结构和算法，熟练掌握 Java 语言，对数据结构有深入的理解，有好的编程习惯；
  • 4年以上 Android 开发经验，至少参加过一个完整的商业级手机应用开发项目；
  • 精通 Android 开发平台及框架原理；
  • 对面向对象设计开发有深入的理解；
  • 有 sdk 相关经验优先；
  • 有 JavaScript 经验的优先。
